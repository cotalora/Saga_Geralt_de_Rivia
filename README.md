# Saga_Geralt_de_Rivia - Api conectada a ChatBot de Telegram implementado con DialogFlow

## Descripción
Api desarrollada para comunicarse con Bot de DialogFlow el cual se encuentra conectado a bot de Telegram. Este muestra información relacionada con los libros de Geralt De Rivia.
